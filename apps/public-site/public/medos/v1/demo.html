📋 Comment l'avoir sur ton site

3 étapes pour intégrer ce widget sur ton site existant (Zahir, Webflow, WordPress, etc.).

1. Whitelister ton domaine

Dans app.cimolace.space → admin de ton tenant → ajoute tondomaine.com dans la liste des domaines autorisés à embarquer le widget.

2. Activer MEDOS sur ton tenant

Même panneau → coche les moteurs MEDOS dont tu as besoin (EHR, Notes SOAP, Prescriptions, Journal santé, etc.). C'est instantané.

3. Coller le snippet ci-dessous dans ton HTML

Sur n'importe quelle page de ton site, à l'endroit où tu veux voir apparaître le widget. C'est tout — pas de build, pas de framework.

🔒 Sécurité automatique : Cimolace vérifie ton domaine via CORS (l'Origin HTTP), délivre un JWT court de 15 min, et applique le RBAC patient. Aucune clé secrète n'est exposée côté navigateur.

{ } Snippet HTML à copier

Colle ces 6 lignes là où tu veux voir apparaître le widget MEDOS. Remplace data-tenant par ton propre slug.

💡 Astuce : Les attributs data-mode disponibles : patient-portal, consent-form, health-tracker, intake-form, appointment-booker. Tu peux mettre plusieurs widgets différents sur la même page.
