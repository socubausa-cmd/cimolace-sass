LIRI Embed — Demo

Widget LIRI Live

Intégrez un live vidéo LIRI dans n'importe quel site — WordPress, Wix, HTML statique, ou votre propre app.

1

Méthode auto-mount (recommandée) — Copier/coller

Copiez ces 2 lignes dans n'importe quelle page HTML. Le widget se monte automatiquement sur chaque élément .liri-live. 
<!-- 1. Conteneur du live -->
<div
  class="liri-live"
  data-tenant="isna"
  data-session="bb775442-aa29-4feb-b7ad-ab2f8e80a3f0"
  data-height="560"
  data-theme="dark"
></div>

<!-- 2. Script widget (1 seule ligne) -->
<script src="https://app.prorascience.org/liri-widget.js" async></script>

Méthode API JS — Contrôle avancé

<div id="mon-live"></div>
<script src="/liri-widget.js"></script>
<script>
  const widget = LiriWidget.mount({
    container: '#mon-live',
    tenant:    'isna',
    session:   'bb775442-aa29-4feb-b7ad-ab2f8e80a3f0',
    height:    560,
    theme:     'dark',

    // Callbacks événements
    onJoined: (info) => {
      console.log('✅ Rejoint la salle :', info.room);
    },
    onEnded: () => {
      console.log('🔴 Session terminée');
    },
    onError: (msg) => {
      console.error('❌ Erreur LIRI :', msg);
    },
  });

  // Démonter proprement si besoin
  // widget.unmount();
</script>

5

Sécurité — Comment ça fonctionne

1. Vérification d'origine
Le widget envoie l'header Origin automatiquement. L'API vérifie que ce domaine est dans la liste blanche du tenant (table tenant_domains).
2. Token embed JWT (30 min)
L'API émet un JWT signé avec LIRI_EMBED_JWT_SECRET contenant {tenant_id, session_id, role, origin, iss}. Sans ce token, impossible de rejoindre la salle.
3. Token LiveKit (1h)
Après validation du JWT embed, l'API génère un token LiveKit viewer (canPublish: false, canSubscribe: true). Le viewer peut regarder mais pas prendre la parole.
4. Isolation multi-tenant
La room LiveKit est nommée isna_bb775442… — prefixée par le slug tenant. Impossible d'accéder aux rooms d'un autre tenant même avec un token valide.
